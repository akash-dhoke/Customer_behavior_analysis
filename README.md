Customer Shopping Behavior Analysis

Overview

This project provides an end-to-end data analytics solution to investigate customer shopping behavior using transactional data from 3,900 purchases. The core objective is to uncover actionable insights into customer demographics, spending patterns, subscription behavior, and product preferences to drive strategic business growth.  The workflow spans the entire data lifecycle: Python for data cleaning and exploratory data analysis (EDA) , PostgreSQL for deep-dive database querying , Power BI for interactive visual reporting, and Gamma for executive presentation generation.  
Dataset Summary
The project utilizes a dataset capturing comprehensive customer transaction records. 
Size: 3,900 rows, 18 columns  
Key Features: * Demographics: Age, Gender, Location, Subscription Status   
Purchase Details: Item Purchased, Category, Purchase Amount, Season, Size, Color   
Behavioral Data: Discount Applied, Previous Purchases, Frequency of Purchases, Review Rating, Shipping Type   
Tech Stack & ToolsData Processing & EDA: Python (Pandas, NumPy)   
Database Management: PostgreSQL  (Compatible with MySQL / SQL Server)  
Data Visualization: Power BI   
Presentation & Reporting: Gamma App (AI-assisted slide decks), Markdown
